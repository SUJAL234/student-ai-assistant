# Student AI Assistant

## Problem
Students struggle with notes, assignments, deadlines, and exam preparation.

## Solution
An AI-powered assistant that helps students manage studies, generate notes, create schedules, track progress, and prepare for exams.

## Features
- AI Chat Assistant
- Smart Notes Generation
- Study Planner
- Assignment Tracking
- Exam Preparation
- Progress Analytics

## Tech Stack
- AI/LLM
- React Native
- Node.js
- MongoDB
